PASSWORD STRENGTH METER BEHAVIOR:
================================================================================

STRENGTH LEVELS:

1. WEAK (Red) ██░░░░░░░░░░░░░░░░░░░░░░
   When: 1-2 criteria met
   Message: "Weak - Add numbers and special characters"
   Examples: "pass" "Pass" "123"

2. FAIR (Orange) ██████░░░░░░░░░░░░░░░░░░░
   When: 2-3 criteria met
   Message: "Fair - Add uppercase letters"
   Examples: "pass123" "password"

3. GOOD (Yellow) ████████████░░░░░░░░░░░░
   When: 3-4 criteria met
   Message: "Good - Consider special characters"
   Examples: "Pass123" "Password1"

4. STRONG (Green) ████████████████████████
   When: 4+ criteria met
   Message: "Strong password!"
   Examples: "P@ss123" "MyPass!123XYZ"

CRITERIA EVALUATED:
✓ Length >= 6 characters
✓ Length >= 10 characters
✓ Length >= 14 characters
✓ Contains lowercase (a-z)
✓ Contains uppercase (A-Z)
✓ Contains numbers (0-9)
✓ Contains special chars (!@#$%^&*)

─────────────────────────────────────────────────────────────────────────────

                                [Error Text: RED] "Please enter [field]"

                                "Password must be at least 6 characters"

                                "Passwords do not match"

                                [Error Text: HIDDEN]

                                "Network error: [details]"

                                [Button: Disabled]

API success                 →   ✓ Alert (Green)
                                "Registration successful!"
                                (Auto-dismisses in 5 seconds)

API error                   →   ✗ Alert (Red)
                                [Specific error message]
